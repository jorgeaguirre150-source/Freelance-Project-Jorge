# Write Proposal

Draft one proposal (max ~90 words) tailored to the given job.

## Inputs
Job title, company, location/region, and description (paste or from `find-clients`).

## Language rule (decide per job)
- Spain / LatAm, or text in Spanish → write in **Spanish**.
- Anywhere else / text in English → write in **English**.

## Structure
1. One opening line naming the company + their specific need/pain.
2. 2–3 bullets of relevant value using REAL proof points:
   - Co-owner of Mercedes-Benz mAzure HUB (multi-region Azure hub-spoke).
   - Principal architect of "Minerva", their NVIDIA AI factory (GPU platform).
   - 30% AWS cost cut via Terraform/IaC.
   - Multi-tenant Kubernetes (AKS/EKS), Agentic AI/RAG, regulated (GxP/HIPAA).
3. Close with ONE question that invites a reply.

## Rules
- Confident, specific, zero fluff. Match the job's stack/keywords.
- **Never state the day-rate first.** If asked: "Mi rango para este scope va entre X e Y; depende del alcance." (anchor 450–800 EUR/day).
- Offer 1–2 variants (e.g., direct-client vs recruiter) if useful.
- Output the proposal ready to copy-paste; do not send anything.
